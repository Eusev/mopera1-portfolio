System Components

In this section, the developers outlined the infrastructure components used in implementing the information, loan and membership system. This included the web hosting, database hosting, and network requirements necessary for the system’s operation. The system relied on a web-based infrastructure, with the following core components:

Web Hosting - the developers used SOMEE to host the system’s web application, making it accessible through browsers and mobile devices.

Database Hosting - The database utilized MySQL for efficient data storage and retrieval, hosted within SOMEE’s environment.

Network Requirement - A stable internet connection was vital for users to access the web service and ensure the information system functions properly.

Software Design

This section outlined the programming languages, framework and technologies used in the development and functionality of the information, loan and membership system. The core components for the web application were as follows:

Programming Languages - the system was developed using HTML, CSS and JavaScript for the front-end. C# was used for the back-end code.

Visual Studio - the developers chose Visual Studio as the main integrated development environment (IDE) for programming the information system, with C# as the programming Language.

Web Framework - the developers used ASP.NET, a web framework developed by Microsoft for building web applications using .NET technologies.

Database System - the developers chose MySQL for its reliability, performance and capability with web application. At the same time, it was a database system that the developers had prior experience.
